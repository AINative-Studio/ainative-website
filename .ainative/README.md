# AINative Studio Configuration

This directory contains AINative Studio-specific configurations for agent orchestration, workflows, and context management.

## Structure

- `agents/` - Agent definitions and configurations
- `workflows/` - Workflow definitions for parallel agent execution
- `memory/` - Persistent memory and context storage
- `context/` - Project context and knowledge base

## Purpose

This folder enables advanced AI-native development workflows including:
- Parallel agent execution
- TDD/BDD workflows
- Context-aware development
- Persistent agent memory
